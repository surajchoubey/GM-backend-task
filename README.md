## Instruction:

```

cd GM-backend-task/
npm install
npm run start
```

To test the application you can refer to this [link](https://drive.google.com/file/d/11ilPtmrJi0lYeSTOB3B3j0PVJKf6uf-S/view)
and send GET / POST request appropriately.
